  <action>
Create `src/components/TaskSprite.tsx` that renders a single task as a glowing billboard circle.

**Props:**
```ts
interface TaskSpriteProps {
  task: Task;
  position: [number, number, number];
}
```

**Implementation:**

1. **Color derivation:**
   - Extract the task's first tag from `task.tags` (array or null).
   - Check if that tag is a valid `TagCategory` key in `TAG_COLORS`.
   - If yes, use `TAG_COLORS[tag]` as the base hue.
   - If no tags or unknown tag, use a default soft blue-white (`#7c8db5`).
   - Shift toward ethereal: Create a `THREE.Color(baseHex)` then apply `.lerp(new THREE.Color('#c8d6e5'), 0.25)` to desaturate slightly toward cool gray. Then `.multiplyScalar(SCENE_CONSTANTS.spriteEmissiveMultiplier)` to push values > 1.0 for bloom.
   - Memoize the computed color with `useMemo` keyed on the tag.

2. **Size variation by drift:**
   - Base radius: `SCENE_CONSTANTS.spriteBaseRadius`
   - If `task.driftCount > 0`, increase radius slightly: `baseRadius * (1 + Math.min(task.driftCount, 5) * 0.06)` — subtle, max ~30% larger at 5 drifts.
   - Memoize with `useMemo`.

3. **Mesh structure:**
   - Use drei `<Billboard>` as the outer wrapper (always faces camera).
   - Inside: `<mesh>` with `<circleGeometry args={[radius, 32]} />`.
   - Material: `<meshBasicMaterial color={glowColor} toneMapped={false} transparent opacity={SCENE_CONSTANTS.spriteOpacity} />`.
   - `toneMapped={false}` is CRITICAL — without it, color values get clamped to 0-1 and bloom will not trigger.

4. **No animation:** Static sprites. No shimmer. Research confirmed shimmer conflicts with `frameloop="demand"`. Static is the design decision from CONTEXT.md.

5. **Position:** The `position` prop is applied to a `<group>` wrapping the Billboard, NOT to the Billboard itself.

**Do NOT:**
- Use `<sprite>` + `<spriteMaterial>` — limited material control, prefer Billboard + mesh per research.
- Use `meshStandardMaterial` — meshBasicMaterial is simpler and works perfectly for bloom.
- Add any animation or useFrame hooks — sprites are static.
- Add click handlers — interactivity is Phase 6.
